Tech Stack

FrontendVanilla HTML5 + CSS3Zero build tools, fast load, easy to maintain

Information Architecture

index.html          ← Talk listing (upcoming + past)
├── claude-talk.html     ← Talk detail + RSVP link
├── speakers.html        ← Speaker profiles
├── submit-talk.html     ← Speaker proposal form  (→ Formspree + localStorage)
├── rsvp.html            ← Attendee RSVP form     (→ Formspree + localStorage)
├── admin.html           ← Review proposals       (← localStorage['submissions'])
├── attendance.html      ← Attendance analytics   (← localStorage['rsvps'])
└── architecture.html   ← This page

Form Contracts (API Equivalent)

Talk Submission — submit-talk.html

FieldTypeRequiredDescription
titletextYesTalk title
abstracttextareaYesTalk summary (max 500 chars)
speaker_nametextYesSpeaker full name
speaker_emailemailYesSpeaker contact email
speaker_biotextareaNoShort speaker bio
preferred_datedateNoPreferred Tuesday for the talk

Destination: POST https://formspree.io/f/YOUR_SUBMISSION_FORM_ID

localStorage key: submissions — array of objects with the fields above plus status (Pending / Approved / Rejected) and submittedAt timestamp.

RSVP — rsvp.html

FieldTypeRequiredDescription
nametextYesAttendee full name
emailemailYesAttendee email (receives confirmation)
talkselectYesWhich talk they are registering for

Destination: POST https://formspree.io/f/YOUR_RSVP_FORM_ID

localStorage key: rsvps — array of objects with the fields above plus registeredAt timestamp.

Deployment Pipeline

git push origin main
  └─► GitHub Actions: .github/workflows/static.yml
        └─► Deploy static files to GitHub Pages
              └─► Live at tuesdaytalk.github.io

No build step. Files are served as-is. Average deploy time: ~30 seconds.
